# Sagar Attendance

A personal web app to track student attendance from the AEC (Aditya college of engineering and technology, Surampalem, Andhra Pradesh) portal with real-time push notifications.

> ⚠️ **This is a personal project. It is not open for contributions or external use.**

> ⚠️ **currently this Application is shutted down(Hibernating) due to Server maintanance costs.**
---

## Technologies Used

- **Node.js** — Server runtime
- **Express** — Web framework
- **Playwright (Chromium)** — Headless browser for attendance scraping
- **Firebase Firestore** — Cloud database for storing user data and notification history
- **Web Push (VAPID)** — Browser push notifications
- **node-cron** — Scheduled attendance polling
- **Docker** — Containerised deployment

---

## How to Use the App

1. **Open the app** in your browser.
2. **Enter your roll number** in the input field and submit.
3. The app displays:
   - **Today's attendance** — subjects and hours attended so far today.
   - **Yesterday's attendance** — subjects and hours attended yesterday.
   - **Overall attendance** — cumulative percentage across all subjects.
4. **Enable notifications** (when prompted) to receive an automatic push notification whenever your attendance is updated during college hours (Mon–Sat, 10 AM – 6:30 PM IST).
5. **View notification history** — tap the notifications icon to see all past attendance alerts.
